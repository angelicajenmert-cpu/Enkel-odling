Enkel odling - professionell statisk webbplats

Filstruktur:
- index.html
- produkt-1.html
- produkt-2.html
- nyhetsbrev.html
- kontakt.html
- thank-you.html
- style.css
- script.js

Publicering:
1. Packa upp ZIP-filen.
2. Ladda upp hela mappen till GitHub.
3. Publicera via GitHub Pages eller anslut repository till Netlify.
4. Startsidan måste heta index.html.

För GA4:
Klistra in Google tag i <head> på varje HTML-sida där kommentaren GA4-FÖRBEREDELSE finns.

För HubSpot/Mailchimp:
Ersätt de befintliga HTML-formulären med embed-kod där platshållarkommentarerna finns.
